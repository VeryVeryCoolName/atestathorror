# Documentația
## Atestatului horror
Acest proiect a fost făcut folosing 2 unelte principale
- [Godot Engine (pentru a programa jocul)](https://godotengine.org/)
- [Github (pentru a organiza jocul)](https://github.com/)

## Structura proiectului
Proiectul este organizat sub forma unui proiect Godot, având structura următoare:
>atestathorror(folder principal)
>  -/Assets - conține mai multe poze/texturi pentru joc
>  -/Music/sfx - conține muzica jocului
>  -/Prefabs - atacurile inamicului
>  -/Scenes - conține scenele jocului - meniuri, nivelul, personaje
>  -/Scripts - conține scriptrile ce controlează logica jocului
>  -/Shaders - conține shadere pentru meniul de pauză
>  project.godot - fișierul principal de configrare a proiectului


## Rularea proiectului din sursă
Pentru a insala jocul, prima dată trebuie instalat folderul cu proiectul, folosind git:
